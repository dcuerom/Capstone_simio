# Instrucciones Pitch - Proyecto de Simulación

## PITCH: Optimización de la Panadería mediante Simulación

## Objetivo:

El Pitch tiene una orientación ejecutiva hacia el equipo gerencial del supermercado, donde lo central serán las recomendaciones, pero que deben ser adecuadamente fundamentadas, y con claros indicadores de gestión de respaldo.

Duración: 10-12 minutos - Descuentos de puntaje por exceso o déficit de tiempo.
Cantidad de slides sugeridas: 10-12

## Sugerencia de Relato:

1. Portada
2. Problema de negocio
¿Qué está pasando?
- Alto nivel de quiebres de stock
- Pérdida de ventas
- Impacto en experiencia del cliente

Mensaje clave: “El sistema actual ….”

## 3. Objetivo del estudio

¿Qué se buscó lograr? Mencionar todos los objetivos priorizados

- Determinar capacidad óptima
- Definir política de producción
- Minimizar quiebres con uso eficiente de recursos
- KPI principal: Nivel de servicio / quiebres

## 4. Enfoque de solución

¿Cómo se abordó?

- Simulación de eventos discretos
- Representación completa del sistema:
- producción
- hornos
- inventario
- demanda
1. Cómo funciona el sistema (simplificado)

Mostrar flujo visual

- Producción por etapas
- Horneado por familias
- Enfriado → reposición → venta

## 6. Variables de decisión

Qué fue testeado

- № panaderos
- № hornos
- № máquinas
- Política de producción
- Hora de inicio, etc.
1. Escenario base

Situación actual

- Capacidad actual
- Política actual
- Resultados:
- Alto quiebre / Baja disponibilidad / Uso de recursos
1. Escenarios evaluados

Qué fue probado

- Cantidades de recursos
- Cambios de política
- Combinaciones de cambios
1. Resultados clave

Comparación visual

- Fill rate
- Kg perdidos
- Utilización

Mensaje: “Algunas decisiones tienen mucho más impacto que otras”
10. Cuellos de botella

Qué limita el sistema

- Horno / fermentación / mano de obra

## 11. Recomendación final

Qué hacer

- № óptimo de recursos
- Política recomendada
- Ajuste horario

Mensaje claro: “Esta es la mejor configuración encontrada”
12. Impacto esperado

Valor para el negocio

- ↓ quiebres
- ↑ ventas
- ↑ nivel de servicio
- Uso eficiente de recursos
